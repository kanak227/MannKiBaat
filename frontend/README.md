MannKiBaat frontend (Neo-Brutalist theme)

Quick start:

1. cd frontend
2. npm install
3. npm run dev

Environment:
- Create a .env file with VITE_API_BASE to point to backend (e.g. VITE_API_BASE=http://localhost:5000/api/v1/mannkibaat)

Notes:
- This is a minimal scaffold matching the backend API from the repository. It includes authentication, poll listing, details, vote casting, and poll creation for coordinators.
