MannKiBaat is a modern polling application built for interactive community engagement. It allows for anonymous voting, ensuring privacy while providing coordinators with powerful tools to manage and visualize public opinion.

-   **Anonymous Voting**: Users can participate in polls without revealing their identity.

-   **Real-time Statistics**: Instant visual feedback on poll results.
-   **Responsive Design**: A clean, neo-brutalist interface for all devices.

## ğŸ Getting Started

### Prerequisites
-   Node.js (v14+)
-   MongoDB instance

### Installation

1.  **Clone & Install:**
    ```bash
    git clone https://github.com/kanak227/MannKiBaat.git
    cd MannKiBaat
    cd backend && npm install
    cd frontend && npm install
    ```

2.  **Environment Setup:**
    Configure `.env` in the `backend` and `frontend` folder.

### Running

```bash
# Backend
cd backend && npm run dev

# Frontend
cd frontend && npm run dev
```

Visit `http://localhost:5173` to get started.
